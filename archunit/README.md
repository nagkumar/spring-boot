check my [page](https://sacco-andres.medium.com/)

## Technologies

Here are some of technologies that I used to developed this example:

- Spring Boot - 2.4.1
- Archunit - 0.15.0

gradle tasks --all

gradle build
gradle compileJava
gradle compileTestJava

gradle clean test

gives the report outputs in build/reports/tests/test/index.html
